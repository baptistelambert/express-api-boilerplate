# express-api-boilerplate
A simple starter for an Express API

# Requirements
- npm (or yarn)
- apidoc cli

# Installation
- After you've cloned or downloaded the repository, install dependencies:

`npm install`

- _(Optional)_ Generate doc

`npm run doc`

- Run the app

`node ./bin/www`
